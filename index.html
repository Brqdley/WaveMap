Wave Height Across
Southern California

By: Bradley Nathanson
As someone who has lived near the Southern California coast my entire life, I was curious to know the trends of the wave heights. Using data from CDIP, which is run by Scripps at UCSD, I displayed yearly trends. I decided to show the trends of 2021 for consistency and accuracy.

You can find the code and data I used for this analysis on my GitHub repository: https://github.com/Brqdley/WaveMap/
